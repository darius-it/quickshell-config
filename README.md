# Quickshell Bar Config

My current Quickshell bar config that I daily drive. The workspace widget is to be used with Niri, for that you need to use [quickshell-niri](https://github.com/imiric/quickshell-niri) in order to get the required Niri QML plugin.

<img width="2558" height="46" alt="image" src="https://github.com/user-attachments/assets/cd123209-9229-446c-9b25-bede9dc6c6b9" />

This is currently a bit barebones, but it has everything I need. In the future I might consider making some fancy hover interactions (for example a hidden hover region to reveal the tray), but for now I'm fairly happy with this config.
